# BME280 Sensor Driver Library

Multi-platform sample code for the BME280 digital humidity, pressure, and temperature sensor (I2C Mini Module from ControlEverything.com, SKU: BME280_I2CS).

## Purpose
Provides ready-to-use I2C communication code to read environmental data from the BME280 sensor across multiple embedded platforms and single-board computers.

## Sensor Output
- Temperature (Celsius and Fahrenheit)
- Barometric pressure (hPa)
- Relative humidity (%RH)

## Target Hardware
- BME280 I2C address: `0x76` (118 decimal)
- Designed for the BME280_I2CS I2C Mini Module
